# RepoEncrypt

Инструмент для клиентского шифрования и расшифровки файлов репозиториев GitHub. Работает полностью в браузере без отправки данных на сервер.

## Возможности

- **Шифрование файлов**: Добавляет маркер `Encoder` и шифрует содержимое поддерживаемых форматов с использованием кодовой фразы.
- **Расшифровка файлов**: Автоматически распознает зашифрованные файлы и восстанавливает оригинальное содержимое при вводе правильной фразы.
- **Просмотрщик кода**: Встроенный редактор для просмотра текстовых файлов и изображений.
- **Экспорт в ZIP**: Массовое скачивание всего репозитория с применением шифрования к поддерживаемым файлам.
- **Безопасность**: Данные не покидают браузер.

## Поддерживаемые форматы

**Текстовые/Код**: JS, TS, Python, Java, C/C++, Go, Rust, PHP, HTML, CSS, JSON, YAML, Markdown, SQL, Shell и другие.
**Изображения**: PNG, JPG, GIF, SVG, WebP (отображаются без шифрования).

**Ограничение**: Максимальный размер файла — 5 МБ.

## Использование

1. Введите ссылку на публичный репозиторий GitHub.
2. Введите кодовое слово (ключ).
3. Выберите режим:
   - **Зашифровать**: Просмотр файлов, скачивание отдельных файлов с шифрованием или создание зашифрованного ZIP-архива.
   - **Расшифровать**: Просмотр дерева файлов. Файлы с маркером `Encoder` автоматически расшифровываются при открытии.

## Алгоритм шифрования

Используется модифицированный шифр Цезаря с полиалфавитной заменой.
- **Алфавит**: Латиница, Кириллица, цифры, символы программирования.
- **Ключ**: Генерируется на основе хеш-суммы введенной пользователем фразы.
- **Важно**: Без точного совпадения кодовой фразы расшифровка невозможна.

## Установка и запуск

Проект не требует установки зависимостей.

1. Клонируйте репозиторий.
2. Разместите файлы (`index.html`, `styles.css`, `app.js`) в корне.
3. Активируйте GitHub Pages в настройках репозитория.
4. Сайт будет доступен по ссылке `https://<username>.github.io/<repo>`.

## Структура проекта

- `index.html` — разметка интерфейса.
- `styles.css` — стили и адаптивный дизайн.
- `app.js` — логика приложения, работа с API GitHub, криптография и управление архивами.
